<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser localStorage -> app | JWT stored in localStorage (D-08, XSS-exfiltration tradeoff accepted) |
| axios -> /api (Vite proxy) | Same-origin relative calls; no cross-origin/CORS surface |
| 401 response -> client session | Expired/invalid token must not leave a stale session |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | stale token after expiry | mitigate | 401 interceptor clears token + redirects /login (D-07) so no expired token lingers |
| T-03-02 | Tampering | XSS via rendered user input (nama) | mitigate | React JSX default escaping; no dangerouslySetInnerHTML (RESEARCH Security Domain) |
| T-03-03 | Information Disclosure | JWT in localStorage | accept | D-08 explicitly accepts localStorage over httpOnly cookies for this course-project scope (documented tradeoff) |
| T-03-04 | Information Disclosure | CORS misconfig masking proxy bugs | mitigate | axios baseURL is relative "/api" via Vite proxy; no CORSMiddleware added (D-16, Pitfall 4) |
| T-03-SC | Tampering | npm installs (all [ASSUMED]) | mitigate | Blocking-human checkpoint (Task 1) confirms npm install against the real registry |
</threat_model>
